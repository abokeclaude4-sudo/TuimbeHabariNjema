# Tuimbe Habari Njema

A React Native / Expo mobile app for organizing Catholic Swahili hymns by category.

## Current Features

- Home screen
- Table of contents
- Hymn categories
- Dynamic category navigation
- Song list display

## Built With

- React Native
- Expo
- Expo Router
- TypeScript

## Planned Features

- Song details page
- Full lyrics
- Search functionality
- Christmas songs
- Kwaresma songs
- Favorites
